# AI Spend Audit


**Instantly audit your startup's AI tooling spend and uncover 30-70% savings opportunities.**

No login. Enter your stack → Get personalized recommendations → Save thousands monthly.

[Start Free Audit →](/audit)

## 🚀 Features

- **AI Stack Analysis** - Cursor, Copilot, Claude, ChatGPT, APIs + more
- **Deterministic Audit Engine** - Rule-based, no AI hallucinations
- **Real Savings Math** - Current vs recommended costs calculated precisely
- **Shareable Reports** - Public URLs with OG previews
- **Credex Integration** - $500+/mo savings → instant consultation
- **Production Quality** - Next.js 15, Supabase, Tailwind, shadcn/ui

## 🎯 How It Works

Enter your AI tools, plans, seats, spend
↓

Engine analyzes pricing + usage patterns
↓

Get instant recommendations + savings calc
↓

Share report or book Credex consultation


## 🛠️ Tech Stack

**frontend**

React 19 — UI framework
Vite — build tool and dev server
Tailwind CSS — utility-first styling
shadcn/ui — pre-built component library (buttons, cards, forms, etc.)
Framer Motion — animations
Recharts — bar charts on the results dashboard
Zustand — client-side state management (persists form progress to localStorage)
React Query (TanStack Query) — data fetching and caching
Wouter — lightweight client-side routing
React Hook Form + Zod — form validation

**Backend**

Node.js 24 — runtime
Express 5 — HTTP server and routing
Zod — request/response validation
Pino — structured JSON logging

**Database**

PostgreSQL — primary database


## 🚀 Quick Start

```bash
# Clone & install
git clone https://github.com/CyberBadal/Ai-For-All.git
cd Ai-For-All
npm install


# Dev
npm run dev

# Build & deploy
npm run build
vercel deploy
```
